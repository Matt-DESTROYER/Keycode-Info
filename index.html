Keycode Info

Press a key to get its info (event type is keydown).

How to sense a key/s down:

.addEventListener("keydown", function(e) {
	e = e || .event;
	// Add scripts here
});

How to sense a key/s up:

.addEventListener("keyup", function(e) {
	e = e || .event;
	// Add scripts here
});

How to sense a key/s press:

.addEventListener("keypress", function(e) {
	e = e || .event;
	// Add scripts here
});
