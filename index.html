Keycode Info

Press a key to get its info (event type is keydown).

How to sense a key/s down:

.addEventListenter("keydown", function(e) {
	e = e || .event;
	// Add scripts here
});

.addEventListenter("keyup", function(e) {
	e = e || .event;
	// Add scripts here
});
